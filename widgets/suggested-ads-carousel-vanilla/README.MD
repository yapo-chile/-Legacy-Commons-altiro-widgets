# Vanilla Listing Item
This Custom Element is created using webpack 4 and Custom Elements v1.

The main goal is support Chrome, Firefox and Safari >= 10.

The code linting is made with Standard JS.

## Instalation

`yarn`

## Dev Server

`yarn dev`

## Build

`yarn build`
